# Beginner Coding Mistakes – Blog Website
This is a simple blog website created as part of a learning project.
The main idea of this site is to share common mistakes that beginners make while learning coding, based on real learning experiences.
This project focuses on understanding how basic web technologies work, along with Google authentication using Firebase.

## Features
- Google Sign-In authentication  
- Login-protected pages  
- Home page with default blog posts visible to all users  
- Option for users to create their own blogs  
- Blogs created by users remain in their own browser  
- Clean and beginner-friendly interface  

## Technologies Used
- HTML  
- CSS  
- JavaScript  
- Firebase Authentication  
- GitHub Pages
- 
## How the Website Works

- Users log in using Google authentication  
- Default blogs are visible to all logged-in users  
- Users can create personal blog posts  
- User-created blogs are stored locally in the browser  
- Logout redirects users to the login page  

## Project Purpose

This project was built to practice frontend development, understand authentication, and create a meaningful beginner-focused blog platform.
